# Equipo Antigravity: Flujo de Trabajo

Este workflow coordina los 4 perfiles expertos para ejecutar proyectos de alta gama.

## Pasos para el Agente

1. **Fase de Conceptualización**
   - Invoca al **Diseñador Anti-Genérico** para definir la identidad visual disruptiva.
   - Consulta al **Especialista en Activos Generativos** para planificar que imágenes o texturas únicas se necesitan.

2. **Fase de Estrategia**
   - El **Arquitecto de Conversión** revisa el concepto para asegurar que el flujo de usuario lleve al objetivo (venta/registro).
   - Se refinan los copies y los CTAs.

3. **Fase de Implementación**
   - El **Creative Developer** construye los cimientos usando React/Vite/Tailwind (según el caso).
   - Aplica CSS avanzado y optimiza para que la carga sea instantánea.

4. **Fase de Pulido**
   - Se integran las micro-interacciones.
   - El **Arquitecto de Conversión** valida la accesibilidad y el SEO.
   - El **Diseñador Anti-Genérico** da el visto bueno final a la estética "Premium".

## Cómo usar este equipo

Simplemente pídele al agente: "Activa el equipo Antigravity para [una tarea]" o "Pásale este componente al Creative Developer para que lo mejore".
